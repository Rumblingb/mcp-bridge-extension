## Detailed Description
MCP Bridge connects you to the Model Context Protocol ecosystem — the open standard that lets AI assistants like Claude, Cursor, and Copilot use external tools.

🔌 **Browse** 24+ MCP servers across categories: databases, payments, cloud, search, and more
📦 **One-click install** — copy MCP config to clipboard, ready to paste into your AI client
⚡ **Zero config** — no terminal, no npm installs, no JSON editing
🔐 **Free tier** — 3 MCP installs free. Pro unlocks unlimited

Built for developers who use AI assistants daily and want to expand their capabilities instantly.
